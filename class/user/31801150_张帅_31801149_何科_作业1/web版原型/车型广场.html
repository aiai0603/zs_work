2020年CACSI成绩公布！北京现代第7次蝉联售后服务满意度第一名

10月15日，第四届中国汽车质量论坛暨2020年汽车用户满意度测评结果新闻发布会在北京落幕。在售后服务满意度指数测评中，北京现代以77分的成绩荣获合资品牌冠军，同时，新一代ix35、全新瑞纳分别摘得细分市场满意度桂冠，再次凸显了北京现代在中国汽车市场全方位领跑的实力。

现代汽车荣登Interbrand全球最佳品牌榜汽车类前五

近日，国际领先的品牌咨询公司Interbrand公布2020全球最佳品牌排行榜，现代汽车的全球品牌价值和汽车厂商榜排名双双高走。评选数据显示，尽管新冠疫情造成市场普遍低迷，现代汽车全球品牌价值却同比逆势上扬1%，达143亿美元，名列汽车品牌第五，全品牌排行榜列第36位。
